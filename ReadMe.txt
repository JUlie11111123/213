Make sure to install the pygame module with the console to get the full experence of this games, if you dont do this I don't gurentee that the game will work or not. Also install pyttsx3.

Use the following 2 commands in command prompt or console:
python -m pip install pygame
python -m pip install pyttsx3 or pip install pyttsx3
You may have to open up your search browser and download pyttsx3.
